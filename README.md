# Overview

As a software engineering student, I am working on improving my ability to analyze real-world data using programming. In this project, I focused on using Python to explore and understand stock market data by applying basic data analysis techniques.

The dataset used in this project contains historical stock data for NVIDIA (NVDA), including information such as date, opening price, highest price, lowest price, closing price, and trading volume. The dataset was obtained from Kaggle, which provides free datasets for analysis.

The main goal of this project is to practice working with data, using code to answer questions, and understanding how data can be used to find meaningful insights. This helps me build skills that are important for real-world software and data-related applications.

# Data Analysis Results

Question 1: What is the highest closing price of NVIDIA stock?
Answer: By sorting the dataset based on closing prices, the program identified the highest value and the specific date when it occurred.

Question 2: What is the average closing price?
Answer: The average closing price was calculated using an aggregation function, which helps summarize the overall value of the stock during the selected time period.

Question 3: Which day had the highest trading volume?
Answer: The program sorted the data by trading volume and found the day with the greatest number of shares traded, indicating the highest level of market activity.
